# hapi-api

#### # Run
By default, this API runs on endpoint `/books` and on port `9000`<br>
You can access it with `http://{url}:9000/books`

---
#### # Get All Books
* `'GET'`
* `http://{url}:9000/books`

#### # Get Book by ID
* `'GET'`
* `http://{url}:9000/books/{id}`

#### # Edit Book by ID
* `'PUT'`
* `http://{url}:9000/books/{id}`

#### # Delete Book by ID
* `'DELETE'`
* `http://{url}:9000/books/{id}`
